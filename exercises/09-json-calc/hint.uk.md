# Підказка - JSON CALC

* Спочатку перевірте обов'язкові поля (`a`, `b`, `operation`).
* Поверніть `422`, якщо якогось поля немає або тип неправильний.
* Реалізуйте `switch` за `operation`:
  * `add`, `subtract`, `multiply`, `divide`
* Поверніть `400` для:
  * невідомої операції
  * ділення на нуль
