# 10 - 5.5 JSON NESTED

Напишіть програму, яка обробляє вкладені JSON-структури.

## ВИМОГИ

Ваша програма повинна:

* Обробляти `POST /json-nested`
* Очікувати JSON-тіло з полями:
  * `user` (об'єкт)
  * `user.name` (рядок)
  * `user.roles` (масив)
* Повертати JSON:
  * `name`
  * `roleCount`
  * `isAdmin` (`true`, якщо `roles` містить `"admin"`)

## ПРИКЛАД ВХІДНИХ ДАНИХ

```json
{
  "user": {
    "name": "John",
    "roles": ["admin", "editor"]
  }
}
```

## ПРИКЛАД ВИХІДНИХ ДАНИХ

```json
{
  "name": "John",
  "roleCount": 2,
  "isAdmin": true
}
```

## ГРАНИЧНІ ВИПАДКИ

* Відсутнє `user` -> `422`
* Відсутнє `roles` -> `422`
* `roles` не є масивом -> `422`

## ПЕРЕВІРКА

```bash
eu-node-basics verify 10 json_nested.js
```

## НАЛАГОДЖЕННЯ
1. Запустіть рішення вручну:
```bash
node json_nested.js 3000
```
2. Відкрийте відповідний `verify.js` для цієї вправи і подивіться, які саме запити/перевірки він виконує.
3. Повторіть ці запити вручну через `curl` або браузер.
4. Додайте тимчасові `console.log(...)`, якщо потрібно перевірити маршрут, метод, тіло запиту або дані відповіді.
5. Після локальної перевірки знову запустіть `eu-node-basics verify ...`.
