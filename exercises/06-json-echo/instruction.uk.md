# 06 - 5.1 JSON ECHO

Напишіть програму, яка обробляє endpoint для "віддзеркалення" JSON.

## ВИМОГИ

Ваша програма повинна:

* Обробляти `POST /json-echo`
* Парсити тіло запиту як JSON
* Повернути той самий JSON-об'єкт у відповіді
* Встановити заголовок `Content-Type: application/json`
* Повернути статус `200`

## ПРИКЛАД ВХІДНИХ ДАНИХ

```json
{
  "message": "Hello"
}
```

## ПРИКЛАД ВИХІДНИХ ДАНИХ

```json
{
  "message": "Hello"
}
```

## ГРАНИЧНІ ВИПАДКИ

* Невалідний JSON -> повернути `400` з повідомленням `Invalid JSON`
* Відсутнє тіло запиту -> `400`

## ПЕРЕВІРКА

```bash
eu-node-basics verify 6 json_echo.js
```

## НАЛАГОДЖЕННЯ
1. Запустіть рішення вручну:
```bash
node json_echo.js 3000
```
2. Відкрийте відповідний `verify.js` для цієї вправи і подивіться, які саме запити/перевірки він виконує.
3. Повторіть ці запити вручну через `curl` або браузер.
4. Додайте тимчасові `console.log(...)`, якщо потрібно перевірити маршрут, метод, тіло запиту або дані відповіді.
5. Після локальної перевірки знову запустіть `eu-node-basics verify ...`.
