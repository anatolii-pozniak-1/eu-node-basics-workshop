# 03 - 4.3 ECHO ROUTE

Додайте endpoint для повернення значення query-параметра.

## ВИМОГИ

Ваша програма повинна:

* Використовувати модуль Node.js `http`
* Слухати порт, переданий першим аргументом командного рядка
* Обробляти `GET /echo?msg=hello`:
  * Статус: `200`
  * Тіло: plain text, що дорівнює значенню `msg`
  * Якщо `msg` відсутній, повернути порожній рядок

## ПЕРЕВІРКА

```bash
eu-node-basics verify 3 echo_route.js
```

## НАЛАГОДЖЕННЯ
1. Запустіть рішення вручну:
```bash
node echo_route.js 3000
```
2. Відкрийте відповідний `verify.js` для цієї вправи і подивіться, які саме запити/перевірки він виконує.
3. Повторіть ці запити вручну через `curl` або браузер.
4. Додайте тимчасові `console.log(...)`, якщо потрібно перевірити маршрут, метод, тіло запиту або дані відповіді.
5. Після локальної перевірки знову запустіть `eu-node-basics verify ...`.
