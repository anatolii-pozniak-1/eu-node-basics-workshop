# 07 - 5.2 JSON OBJECT

Напишіть програму, яка обробляє властивості об'єкта з JSON-запиту.

## ВИМОГИ

Ваша програма повинна:

* Обробляти `POST /json-object`
* Очікувати JSON-тіло з полями:
  * `name` (рядок)
  * `age` (число)
* Повертати JSON:
  * `greeting`: `"Hello <name>"`
  * `isAdult`: `true`, якщо `age >= 18`, інакше `false`

## ПРИКЛАД ВХІДНИХ ДАНИХ

```json
{
  "name": "Anatolii",
  "age": 30
}
```

## ПРИКЛАД ВИХІДНИХ ДАНИХ

```json
{
  "greeting": "Hello Anatolii",
  "isAdult": true
}
```

## ГРАНИЧНІ ВИПАДКИ

* Відсутнє `name` -> `422`
* Відсутнє `age` -> `422`
* `age` не є числом -> `422`

## ПЕРЕВІРКА

```bash
eu-node-basics verify 7 json_object.js
```

## НАЛАГОДЖЕННЯ
1. Запустіть рішення вручну:
```bash
node json_object.js 3000
```
2. Відкрийте відповідний `verify.js` для цієї вправи і подивіться, які саме запити/перевірки він виконує.
3. Повторіть ці запити вручну через `curl` або браузер.
4. Додайте тимчасові `console.log(...)`, якщо потрібно перевірити маршрут, метод, тіло запиту або дані відповіді.
5. Після локальної перевірки знову запустіть `eu-node-basics verify ...`.
