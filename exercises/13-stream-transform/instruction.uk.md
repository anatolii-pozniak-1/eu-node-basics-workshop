# 13 - 8.3 STREAM TRANSFORM

Трансформуйте вміст файлу під час стрімінгу у відповідь.

## ВИМОГИ

Ваша програма повинна:

* Обробляти `GET /upper?fileName=<name>`
* Читати файл з локального диска (поточна робоча директорія) як стрім
* Перетворювати весь текст у верхній регістр під час стрімінгу
* Повернути статус `200`
* Якщо `fileName` відсутній або файл не існує, повернути статус `400`

## ПЕРЕВІРКА

```bash
eu-node-basics verify 13 stream_transform.js
```

## НАЛАГОДЖЕННЯ

Запустіть сервер вручну:

```bash
echo "Hello Mixed Case" > file.txt
node stream_transform.js 3000
```

Потім перевірте обидва випадки:

```bash
curl -i "http://127.0.0.1:3000/upper?fileName=file.txt"
curl -i "http://127.0.0.1:3000/upper"
```

Перевірте:
* перший запит повертає `200` і текст у верхньому регістрі
* другий запит повертає `400`
