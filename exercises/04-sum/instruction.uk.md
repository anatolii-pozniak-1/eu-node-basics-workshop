# 04 - 4.4 SUM ROUTE

Додайте endpoint для обчислення суми через query-параметри.

## ВИМОГИ

Ваша програма повинна:

* Використовувати модуль Node.js `http`
* Слухати порт, переданий першим аргументом командного рядка
* Обробляти `GET /sum?a=<number>&b=<number>`:
  * Статус: `200`
  * Тіло: JSON `{ "sum": <number> }`
* Якщо `a` або `b` відсутні/некоректні:
  * Статус: `400`
  * Тіло: JSON `{ "error": "Invalid numbers" }`

## ПЕРЕВІРКА

```bash
eu-node-basics verify 4 sum_route.js
```

## НАЛАГОДЖЕННЯ

Запустіть сервер вручну:

```bash
node sum_route.js 3000
```

Потім перевірте обов'язкові випадки:

```bash
curl -i "http://127.0.0.1:3000/sum?a=2&b=5"
curl -i "http://127.0.0.1:3000/sum?a=2"
```

Перевірте:
* чи сервер запустився
* чи використовується `GET /sum` з query-параметрами
* чи повертається JSON і для успішного, і для помилкового випадку
