# Manual HTTP Router Workshopper

A minimal, learnyounode-style practice with atomic exercises for **Manual HTTP Router**.

## Install (local)

```
npm link
```

Then run:

```
manual-http-router
```

## Commands

```
manual-http-router            # interactive menu
manual-http-router list       # list exercises
manual-http-router select 1   # show instructions
manual-http-router verify <number> <path-to-solution>
```

## Exercises
- ROOT ROUTE
- TIME ROUTE
- ECHO ROUTE
- SUM ROUTE
- NOT FOUND ROUTE

## Notes
- No external dependencies.
- Progress is stored in `.progress.json`.
